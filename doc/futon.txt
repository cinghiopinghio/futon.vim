*futon.txt*	For Vim version 8.0	Last change: 2021 May 18


		Futon color scheme    by Cinghio

Futon is a colorless colorscheme inspired by Photon.

==============================================================================
Colorscheme options				*futon-options*

						*g:futon_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:futon_transp_bg = 0
<
 vim:tw=78:ts=8:noet:ft=help:norl:
